\section*{Part 1: Summary of Today's Work (Constraints C1--C4)}

Today's focus was implementing the four core constraint predicates in \texttt{src/constraints.py},
completing the \texttt{expr\_generator} $\rightarrow$ \texttt{metric} $\rightarrow$ \texttt{constraints} pipeline.

\begin{itemize}
    \item \textbf{C1 -- Structural Constraint:} Implemented \texttt{make\_c1\_structural} enforcing
    two sub-rules: (a) maximum nested trig depth of 1 (\texttt{sin(cos(x))} rejected; parallel
    siblings accepted); (b) maximum flattened \texttt{Add}/\texttt{Mul} arity of 3 (2 consecutive
    binary ops at limit). Handled SymPy's automatic flattening by checking \texttt{len(node.args)}.

    \item \textbf{C2 -- Depth Constraint:} Implemented \texttt{make\_c2\_depth} using a recursive
    \texttt{.args}-walk depth. Formally documented the divergence between SymPy depth and generator
    string depth (e.g., \texttt{-x} $\rightarrow$ \texttt{Mul(-1,\,x)}, depth 1). The bias is
    bounded ($\leq +2$ per node) and measured empirically by \texttt{test\_depth\_divergence\_measured}
    (mean gap: 0.42, max: 3). Rejected the passthrough alternative to preserve the
    \texttt{Expr $\rightarrow$ bool} purity contract.

    \item \textbf{C3 -- Operator Whitelist:} Implemented \texttt{make\_c3\_operator} via
    \texttt{preorder\_traversal}, skipping leaf types (\texttt{Symbol}, \texttt{Number}, etc.) and
    rejecting non-leaf nodes outside the permitted frozenset. The single \texttt{\_OP\_CLASS\_MAP}
    dict (Bug~\#4 fix) expands \texttt{"-"} to both \texttt{Add} and \texttt{Mul}.
    Empirically confirmed $\rho(C_3) \approx 0.23$, in the expected 0.3--0.6 band.

    \item \textbf{C4 -- Positivity (Non-Negativity):} Implemented \texttt{make\_c4\_positivity}
    with a vectorised \texttt{lambdify} fast path and a \texttt{.subs()} fallback.
    Predicate: $f(x,y) \geq 0$ (not $> 0$) per Research Plan~\S5; boundary $f = 0$ returns
    \texttt{True}. RNG seed read from config key \texttt{rng\_seed} (Bug~\#2 fix).

    \item \textbf{Testing:} 29 new tests added across \texttt{TestC1Structural},
    \texttt{TestC2Depth}, \texttt{TestC3OperatorWhitelist}, \texttt{TestC4Positivity},
    and \texttt{TestBuildConstraints}. Includes mandatory boundary-locking tests and the
    empirical divergence measurement for Handbook~\S12.
    Full suite: \textbf{60 tests, all passing}.

    \item \textbf{Integration Smoke Test:} Running the full estimator with
    \texttt{build\_constraints(cfg)} at $N=1000$ returned
    $\rho = [0.404,\; 0.987,\; 0.227,\; 0.231]$ --- all within the expected ranges.

    \item \textbf{Documentation:} Updated \texttt{DESIGN\_CONTEXT.md} (\S6.2 marked Implemented;
    four new Known Gotchas), \texttt{API.md} (\texttt{src.constraints} section),
    \texttt{config.yaml} (\texttt{rng\_seed:\,0} + comment), and \texttt{LOGBOOK.md} (Entry~06).
\end{itemize}

    \item \textbf{Append-Only Results:} Each run writes to a new \texttt{run\_id} directory;
    no overwriting of prior results (DESIGN\_CONTEXT~\S6.4).
\end{itemize}
